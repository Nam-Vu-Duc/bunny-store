**BUNNY HOUSE - FULL STACK WEB DEVELOPMENT**
===============================
A cosmetics website that facilitates sales management and customer relationship management (CRM).

**Main Functionalities**
===============================
**User:**
![image](https://github.com/user-attachments/assets/a3a36ae0-ed15-46c0-9fc4-339737611f25)
- Offer multiple tools to find the products you need quickly and easily. With features like Search (by name), Filter (by price, rateNumber, saleNumber) or Sort (by price), and Chat support (with admin or Gemini API).
- Offer account management, including signIn, signUp, resetPassword and update personal information
- Offer order management, including place orders, find orders by id, track orders status, and rate your orders experience after completion.

**Admin:**
![image](https://github.com/user-attachments/assets/a7584591-e60b-4d1e-b41d-565c5c161a1f)
- Build dashboard with Chartjs, export to Excel file with SheetJS
- Offer tools to find products with features like Search, Filter and Sort
- Support essential CRUD operations to process CRM and HRM workflows.
  - Manage customers, orders, suppliers, and purchases efficiently within the CRM module
  - Manage stores and employees seamlessly in the HRM module.


**Tech Stack**
===============================
- FrontEnd: HTML, CSS, JS
- BackEnd: NodeJS, Express, MongoDB, Cloudinary
- Library: WebSocket, ChartJs, SheetJs, ...
